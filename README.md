# Go Roadmap

This repository documents my journey learning Go.

The goal is to build a solid understanding of the language by studying its fundamentals, writing examples, and documenting what I learn throughout the process.

Instead of focusing only on building projects, this repository is intended to consolidate core concepts that are essential for backend development.
